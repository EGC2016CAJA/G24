OpenSSL PHP Extension
PDO PHP Extension
Mbstring PHP Extension
Tokenizer PHP Extension
XML PHP Extension

# Descripción del proyecto
Nuestro módulo se encarga de gestionar el almacenamiento de votos de la aplicación.
